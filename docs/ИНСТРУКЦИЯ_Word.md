# Инструкция по оформлению отчёта КТ-1 в Word

Отчёт сгенерирован скриптом `generate_docx.py` и сохранён как `КТ1_отчет.docx`.

## Форматирование (согласно требованиям)

- **Шрифт:** Times New Roman, 12 pt
- **Межстрочный интервал:** 1.5
- **Абзацный отступ:** 1.25 см
- **Выравнивание:** по ширине
- **Поля:** левое 25 мм, правое 10 мм, верхнее и нижнее 20 мм

## Что нужно сделать вручную

1. **Нумерация страниц**
   - Вставка → Номер страницы → Внизу страницы → Простой номер 2 (по центру)
   - Двойной клик по колонтитулу → включите «Особый колонтитул для первой страницы», чтобы титульник был без номера

2. **Заполнить плейсхолдеры на титульном листе**
   - \<Название университета\>
   - \<код и название\> (направление подготовки)
   - \<Фамилия И.О.\> (студент)
   - \<номер группы\>
   - \<Фамилия И.О., учёная степень\> (научный руководитель)
   - \<Город\>

3. **Проверить соответствие шаблону**
   - Сравните титульный лист с файлом «Программный проект ПМИ - Титульный лист к Отчету по КТ1 (1).docx» и при необходимости скорректируйте оформление.

## Пересоздание документа

```bash
py -3 generate_docx.py
```
