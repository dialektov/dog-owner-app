# DogPaw — Помощник владельца собаки

Приложение для iOS и Android: дневник питомца, трекер прогулок, карта мест, социальная сеть для собаководов.

## Структура репозитория

```
dog-owner-app/
├── backend/          # REST API на Go (Gin, GORM, SQLite)
├── src/              # Мобильное приложение (React Native + Expo)
├── assets/           # Иконки, сплэш
├── docs/             # Документация, отчёты, инструкции
├── eas.json          # Конфигурация EAS Build/Submit для магазинов
└── README.md
```

## Быстрый старт

### Бэкенд (Go)

```bash
cd backend
go mod tidy
go run .
```

API: `http://localhost:8080`

Демо-данные: `go run ./cmd/seed`

### Мобильное приложение

Проект использует **Expo SDK 54** (совместим с актуальной версией Expo Go на устройстве).

```bash
npm install
npx expo install --fix
npm start
```

Откройте в Expo Go по QR-коду или нажмите `a`/`i` для эмулятора.

> Настройте `src/api/config.ts` — укажите IP компьютера для теста на реальном устройстве.

## Функции

| Функция | Описание |
|---------|----------|
| **Карточка питомца + QR** | Анкета собаки, QR на ошейник (потеря / знакомство) |
| **Трекер прогулок** | Километры, маршрут, калории |
| **Калькулятор кормления** | Норма корма, напоминания |
| **Карта мест** | Ветклиники, парки, зоомагазины, кафе |
| **Умная карта** | Где гуляют друзья, статусы |
| **Друзья и лента** | QR-знакомства, чаты, посты |
| **База знаний** | Статьи по воспитанию, уходу, здоровью |

## Публикация в App Store и Google Play

Чтобы выложить приложение в **App Store** и **Google Play**, используется [EAS (Expo Application Services)](https://docs.expo.dev/build/introduction/): облачная сборка и отправка в магазины. В проекте уже есть конфигурация `eas.json`.

Подробная инструкция: **[docs/ПУБЛИКАЦИЯ_В_МАГАЗИНЫ.md](docs/ПУБЛИКАЦИЯ_В_МАГАЗИНЫ.md)** — что понадобится (аккаунты Apple Developer и Google Play), команды `eas build` и `eas submit`, настройка и альтернатива с ручной загрузкой билда.
