# DogPaw — Помощник владельца собаки

Приложение для iOS и Android: дневник питомца, трекер прогулок, карта мест, социальная сеть для собаководов.

## Структура репозитория

```
dog-owner-app/
├── backend/          # REST API на Go (Gin, GORM, SQLite)
├── src/              # Мобильное приложение (React Native + Expo)
├── assets/           # Иконки, сплэш
└── README.md
```

## Быстрый старт

### Бэкенд (Go)

```bash
cd backend
go mod tidy
go run .
```

API: `http://localhost:8080`

Демо-данные: `go run ./cmd/seed`

### Мобильное приложение

```bash
npm install
npm start
```

Откройте в Expo Go по QR-коду или нажмите `a`/`i` для эмулятора.

> Настройте `src/api/config.ts` — укажите IP компьютера для теста на реальном устройстве.

## Функции

| Функция | Описание |
|---------|----------|
| **Карточка питомца + QR** | Анкета собаки, QR на ошейник (потеря / знакомство) |
| **Трекер прогулок** | Километры, маршрут, калории |
| **Калькулятор кормления** | Норма корма, напоминания |
| **Карта мест** | Ветклиники, парки, зоомагазины, кафе |
| **Умная карта** | Где гуляют друзья, статусы |
| **Друзья и лента** | QR-знакомства, чаты, посты |
| **База знаний** | Статьи по воспитанию, уходу, здоровью |
